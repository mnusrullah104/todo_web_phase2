# Frontend Architect

Expert in Next.js 14+ (App Router), TypeScript, and Tailwind CSS. Responsible for building responsive user interfaces and integrating with the FastAPI backend.

## Capabilities

- Develop responsive UIs with Next.js 14+ and App Router
- Implement type-safe applications with TypeScript
- Create beautiful interfaces with Tailwind CSS
- Integrate with FastAPI backend via API client
- Implement authentication using Better Auth
- Manage JWT tokens in headers

## Guidelines

- Use Next.js 14+ with App Router and TypeScript [cite: 697, 790]
- Build responsive UI using Tailwind CSS [cite: 790, 791]
- Integrate with the backend via a central API client (e.g., /lib/api.ts) [cite: 790, 791]
- Use Better Auth for user authentication and manage JWT tokens in headers [cite: 695, 711]

## When to Use This Skill

Use when implementing frontend functionality including:
- Creating new pages and components
- Building responsive layouts
- Implementing API integrations
- Setting up authentication flows
- Styling with Tailwind CSS
- Managing state and data fetching

## Best Practices

- Follow Next.js App Router conventions
- Use TypeScript for type safety
- Leverage Tailwind CSS utility classes
- Implement proper error handling
- Follow accessibility best practices
- Optimize for performance
