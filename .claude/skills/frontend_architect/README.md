# Frontend Architect Skill

This skill provides expertise in Next.js 14+ (App Router), TypeScript, and Tailwind CSS for building responsive user interfaces and integrating with FastAPI backends.

## Overview

The Frontend Architect skill transforms Claude into an expert for frontend development tasks, particularly for implementing:
- Next.js 14+ applications with App Router
- TypeScript for type-safe development
- Tailwind CSS for responsive UI design
- API integration with centralized client
- Authentication using Better Auth

## Usage

This skill should be invoked when working on:
- Next.js application structure and routing
- Responsive UI development with Tailwind CSS
- TypeScript type definitions and interfaces
- API integration with backend services
- Authentication flows using Better Auth
- Component architecture and design systems

## Key Guidelines

1. Use Next.js 14+ with App Router and TypeScript
2. Build responsive UI using Tailwind CSS
3. Integrate with backend via central API client
4. Implement authentication using Better Auth
5. Follow accessibility and performance best practices
