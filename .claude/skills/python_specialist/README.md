# Python Specialist Skill

This skill provides expertise in Python 3.13+, FastAPI, SQLModel, and UV for implementing backend logic, RESTful APIs, and database integrations.

## Overview

The Python Specialist skill transforms Claude into an expert for Python backend development tasks, particularly for implementing:
- RESTful APIs using FastAPI
- Database interactions with SQLModel and Neon PostgreSQL
- Proper project structure following clean code principles
- Package management with UV
